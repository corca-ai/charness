# Sequence Discipline

This reference captures the sequence-discipline moves that matter to
`ideation`.

## Core Moves

- decide the upstream question whose answer most changes the rest
- prefer steps that preserve or increase later options
- delay detail that would harden the shape before adjacent structure is known
- keep the artifact order aligned with the order the concept should unfold

## Translation For Ideation

- ask the question that clarifies the rest of the concept, not the easiest one
- separate durable structure from chronology before debating surface detail
- if a direction would close off better alternatives too early, treat it as
  downstream and delay it

# Charness 0.13.5

This patch release closes the reviewer-tier reliability carrier for #275 and
#276.

## Operator Impact

- Installed handoff can load its sibling `issue` skill from the plugin layout
  (`skills/issue`) instead of silently falling back to a source-tree-only path.
- Handoff issue-source diagnostics now survive the parse-to-propose pipeline, so
  live issue lookup failures stay visible to operators.
- Achieve goal activation now blocks when consequential decisions are hidden in
  `Non-Goals`, verification, interview decisions, or critique findings without
  a current `Discuss before activation:` summary.
- Reviewer-tier closeout now records and verifies fresh-eye Act Before Ship
  dispositions before broad verification.

## Update Steps

Run `charness update` to pull 0.13.5, then restart Claude Code or Codex if the
host cache still shows the previous version.

No manual migration is required beyond the normal update flow.

## User-facing behavior change to document

After `charness update` to 0.13.0, shaping a **new** goal (`Created ≥ 2026-05-31`)
whose `## Context Sources` names an external URL refuses the `complete` flip
until a `Gather:` step (or `Gather: n/a — <reason>`) is recorded in
`## Coordination Cues`; same for a release-surface token and a `Release:` step.
Existing/older goals are unaffected (grandfathered). The failure message is
actionable and names the opt-out.

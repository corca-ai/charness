## Findings

### Act Before Ship

- A standalone fresh-eye release critique was required by the publish helper.
  This artifact satisfies that release-scope review gate.

### Bundle Anyway

- Minor (0.21.0) is the lightest honest bump: every commit in `v0.20.0..HEAD` is
  additive or a fix (nose 0.5 advisory adaptation — still `policy: advisory`,
  parser backward-compatible with the 0.4 array; in-process test conversions;
  scaffolds; cleanup). No invocation/CLI/compatibility break forces a major, and
  the delta is more than a single patch fix.
- Real-host proof is NOT triggered for this slice (`check_real_host_proof.py`
  reports `required: false`); no second-machine smoke is owed for 0.21.0.

### Over-Worry

- The announcement cites the ratchet drop `94→90 (55→51 convertible)` — the
  delta attributed to the highlighted five-test item alone. The full-window
  delta is `96→90 (54→51)` because an earlier commit (`0604f3d2`) did `96→94`
  first. The announcement's figure is conservative (under-claims the full-window
  improvement), direction-correct, and not inflated. No change required.
- "236 families" (raw nose 0.5 scan, commit `7bd7d1ee`) vs "~20 families"
  (ranked extractable candidates the inventory surfaces) are different metrics,
  not a contradiction; both verified.

### Valid But Defer

- Promoting nose clone findings from advisory to a hard standing gate was
  explicitly out of scope for this release (goal scope B). Deferred.
